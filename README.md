# .github

Shared GitHub templates and workflows for HersRD repos.

## Issue Templates

| Template | Type | Labels | Description |
|----------|------|--------|-------------|
| [Bug Report](.github/ISSUE_TEMPLATE/bug.yml) | Bug | `bug` | Report a bug to help us identify and fix issues. Includes fields for bug description, steps to reproduce, and optional screenshots. |
| [Research](.github/ISSUE_TEMPLATE/research.yml) | Task | `documentation` | Research and document a new feature, technology, or implementation approach. Includes fields for requirements checklist and deliverable. |
| [Story](.github/ISSUE_TEMPLATE/story.yml) | Feature | — | Create a user story issue to describe a feature or enhancement from the user's perspective. Includes fields for requirements and acceptance criteria as checklists. |

### Configuration

- **Blank issues are disabled** ([config.yml](.github/ISSUE_TEMPLATE/config.yml)) — contributors must use one of the templates above when opening an issue.
