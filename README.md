# .github

Shared GitHub templates and workflows for HersRD repos.

## Issue Templates

| Template | Description |
|----------|-------------|
| [Story](.github/ISSUE_TEMPLATE/story.yml) | Create a user story issue to describe a feature or enhancement from the user's perspective. Includes fields for a user story narrative, acceptance criteria, additional context, and story point sizing. |

### Configuration

- **Blank issues are disabled** ([config.yml](.github/ISSUE_TEMPLATE/config.yml)) — contributors must use one of the templates above when opening an issue.
